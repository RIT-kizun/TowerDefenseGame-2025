# タワーディフェンスゲーム

## 概要

このプロジェクトは、大学の最終課題のために作成したpygameを利用したゲームです。
赤いマスから出てくる敵をコストを利用しブロック、アタックして青いマスに入るのを阻止するゲームです。

## 使い方

1. ダウンロードしたディレクトリ移動し、ターミナルで python3 main.py を実行
2. 操作は全てマウスで行う。

画面下部にあるblockerやattackerをコストを支払って配置し、enemy(hp:4)を全て倒したら勝利です。
blockerは道（灰色）、attackerは高台（白）に配置できます。
attackerは直線攻撃しかできないためクリックして向きを変える必要があります。
配置中のattackerをクリックするごとに反時計回りに90度回転します。
enemyは秒間1の攻撃をしてきます。ずっとブロックできるわけじゃないので注意しましょう。
ユニットの選択を解除したい場合cancelを押しましょう。

3. LIFE(耐久値)が0になるもしくはENEMY（敵総数）が0になり盤面のenemyを全て倒すことができればゲーム終了です。ウィンドウの罰ボタンを押しても終了できます。


## 工夫した点
できるだけキーボードからの入力は避けたかった（ゲームの都合上マウスは必ず持つため）のでクリックだけでできるように実装した。
今どんな状態かをできるだけ画面に表示したかったのでブロック中は敵の見た目を変えたり選択中のユニットを表示したりするようにした。
enemy,blocker,attacker,bullet,下部ボタン全て画像を作っている。
↑pygameで作成しても良かったが今後使うなら画像だと判断したため。
